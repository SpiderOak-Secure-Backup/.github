## What is SpiderOak?

SpiderOak is a zero-knowledge encrypted cloud backup, sync, and sharing platform that ensures no one — not even SpiderOak — can access your data . The platform provides backup and storage capabilities for Windows, Mac, Linux, iOS, and Android devices, plus an innovative data transfer tool, especially for large and highly sensitive files . SpiderOak is trusted by enterprises, government agencies, and organizations that prioritize data privacy and security, supporting compliance with CMMC, ITAR, DFARS, and other rigorous regulations .

Infrastructure teams managing secure backup and file sharing benefit from SpiderOak's zero-knowledge architecture—encryption keys are generated and stored solely on the client side, ensuring complete data privacy . System administrators appreciate the unlimited storage options and comprehensive compliance certifications including FedRAMP, CMMC, ITAR, and EU GDPR .

## Key Features

| Feature | Description |
|---------|-------------|
| **Zero-Knowledge Encryption** | Data is encrypted before leaving your device, and encryption keys are only known to you. SpiderOak has no means to decrypt your data . |
| **Multi-Device Backup** | Backup, sync, and share across Windows, Mac, Linux, iOS, and Android devices from a single account . |
| **SpiderOak One** | Back up and sync files with unlimited storage (standard pricing 1 TB per user) . |
| **CrossClave** | Secure file sharing and collaboration with client-side encryption, privileged access controls, immutable audit logs, and multi-factor authentication . |
| **Send (SpiderOak Transfer)** | Send and request files securely with password-protected links and automatic expiry . |
| **No File Size Limits** | Send files of any size — no file size restrictions for sharing . |
| **Privacy Design** | Preserves file names and folder hierarchy, does not hide metadata from users . |
| **Blockchain Integration** | Optional audit-verification of data integrity using blockchain technology . |
| **Compliance Ready** | Supports CMMC, ITAR, DFARS, and GDPR compliance requirements . |

## Installation Guide

### Option 1: Desktop App (Windows)

**Step 1:** Visit the SpiderOak download page.

**Step 2:** Download the SpiderOak One installer for Windows.

**Step 3:** Run the installer and follow the setup wizard.

**Step 4:** Sign in to your SpiderOak account or create a new account.

**Step 5:** Select the folders you want to backup and sync.

**Step 6:** Click **Next** to complete the setup and start backing up your data.

### Option 2: CrossClave (Enterprise)

**Step 1:** Contact SpiderOak sales for CrossClave deployment options.

**Step 2:** Deploy CrossClave across your organization with centralized administration.

**Step 3:** Configure privileged access controls, audit logs, and security settings.

**Step 4:** Enable secure file sharing and collaboration for teams.

### Option 3: Send (File Transfers)

**Step 1:** Create a free account or log in.

**Step 2:** Select a file to share or use the upload link to request files .

**Step 3:** Set password, expiry, and download limits.

**Step 4:** Share the secure link with recipients—no account required .
